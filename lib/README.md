Shared server-side utilities and client singletons. Not React components. animations.ts: motion/react config objects imported by section components. anthropic.ts: Claude client, server-only, used by /api/chat only. db.ts: Prisma singleton preventing hot-reload multi-instance issues. email.ts: Resend client for direct email if needed alongside n8n. utils.ts: cn() (clsx + tailwind-merge), formatDate(), pure utilities. Dependency flow is one-way: app → components → lib. lib never imports from components/ or app/.
