Primary keyword targets for each page. All targets are specific to professional services in Puerto Rico and Orlando.

## Keyword Strategy Overview

**Core positioning keywords:**
- "AI implementation" (not "AI tools" or "AI software")
- "professional services" (the vertical category)
- "Puerto Rico" + "Orlando" (geographic modifiers)
- Industry terms: "medical practice", "law firm", "CPA firm"

**Intent targeting:**
- Discovery: "AI for [industry] Puerto Rico"
- Problem-aware: "automate [specific task] [industry]"
- Solution-aware: "AI intake automation", "client follow-up automation"
- Offer-aware: "free AI audit", "AI audit professional services"

## Keyword Rules

1. **Do not target generic AI keywords.** "AI tools", "AI software", "artificial intelligence" — too competitive, wrong intent.
2. **Always combine industry + geography.** "law firm automation Puerto Rico" is better than either alone.
3. **Do not claim keyword rankings.** This map is a targeting guide, not a guarantee.
4. **Monitor via Google Search Console** after launch. Adjust based on actual impression data.
